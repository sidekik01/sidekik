# sidekik Product Principles

## Mission

sidekik is the creative Sidekick behind every great video.

Our goal is not to generate captions.

Our goal is to help creators produce better performing videos faster.

## Design Philosophy

Every screen answers one question.

Analyze  
What did I upload?

Transcript  
What was said?

Style  
How should it look?

Director  
How can it perform better?

Export  
Where is this going?

## Product Principles

- Great defaults beat endless settings.
- AI recommends. Users approve.
- Every feature must save time.
- Speed is a feature.
- Never overwhelm.
- Consistency beats novelty.
- Everything should feel calm and premium.
- Professional creators first.
- Beginners should succeed with one click.
- Every interaction should reduce friction.
- The app should feel like an experienced creative director sitting beside you.

## Engineering Principles

- Reusable components.
- Reusable services.
- Reusable business logic.
- Frontend and backend separated.
- Everything strongly typed.
- No duplicated logic.
- Every feature must support future SaaS growth.

## Definition of Done

A sprint is complete only if:

- `npm run build` passes.
- `npm run dev` passes.
- UI is polished.
- Empty states exist.
- Error states exist.
- Loading states exist.
- Feature integrates naturally into the workflow.
